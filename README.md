Integration Project: printf

Description:
Ce projet consiste à recréer une version simplifiée de la fonction printf de la bibliothèque standard C (stdio.h). La fonction _printf produit une sortie formatée sur le flux de sortie standard (STDOUT).

Elle parcourt une chaîne de caractères et remplace les spécificateurs de format (commençant par %) par les arguments correspondants fournis à la fonction.

Installation et Compilation:
Le code est compilé sur Ubuntu 20.04 LTS en utilisant gcc avec les options suivantes :

gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o _printf
